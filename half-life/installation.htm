Installation

 

Transférez le dossier  HALF-LIFE   dans le dossier "Thèmes" du répertoire racine de votre site". Ensuite allez dans votre panneau d'administration et faites la marche nécessaire.

Ensuite, si vous souhaitez remplacer ce qui est écrit sur cette image par le "Nom de votre site et Votre slogan"  :

 

Il vous faut ouvrir le fichier " header.psd" qui est dans le dossier "fichierssources" , faire les modifications nécessaires et enregistrer le fichier sous le nom  index_04.jpg dans le dossier "imagens"

PS: Si vous rencontrez des bugs, merci de nous les faire parvenir ici

Attention: Nous ne pourrons en aucun cas être tenus responsable pour tout problème sur votre site dû à l'installation de ce thème. Il a été testé sans bugs apparents à première vue.
